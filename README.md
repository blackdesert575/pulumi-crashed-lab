# pulumi-crashed-lab
Play pulumi with python3 scripts to make some infra building plan on AWS

## Prerequisites

* An AWS Account
* Linux host such as Arch Linux, Debian, Ubuntu, RHEL ...etc
* docker cli and engine installed
    * [Install Docker Engine](https://docs.docker.com/engine/install/)
* Build a docker container image in tools/pulumi_with_python3

## Quick Start

* Run with docker with volume mount with this git repo 
  or running with docker-compose.yaml via docker compose cli tools
